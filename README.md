## 🚀 Wie man die Website verwendet

1. Öffne `index.html` in deinem Browser
2. Durchsuche die verfügbaren Produkte
3. Klicke auf "Hinzufügen", um Produkte in den Warenkorb zu legen
4. Klicke auf das 🛒 Symbol oben rechts, um den Warenkorb zu sehen
5. Klick auf "Zur Kasse", um die Bestellung abzuschließen
